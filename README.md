Backport of the **Enchanting Infuser** mod to Minecraft **1.12.2**.

## 📖 About

This mod adds the **Enchanting Infuser** — a powerful enchanting table that allows you to:

- Choose exactly which enchantments to apply
- Select the level of each enchantment
- See the required experience cost before enchanting
- Repair items directly in the GUI (Advanced version)

The mod is a backport of the original Enchanting Infuser mod from newer Minecraft versions, bringing its functionality to 1.12.2 modpacks and servers.

## ✨ Features

- **Custom Enchanting GUI** — Select enchantments from a scrollable list
- **Real-time Cost Calculation** — See exactly how much XP you'll spend
- **Search Bar** — Filter enchantments by name
- **Advanced Infuser** — Upgradeable version with item repair functionality
- **Book Power Display** — Shows how many bookshelves are providing power
- **Multiplayer Compatible** — Works on servers

## 📦 Installation

1. Install **Minecraft Forge** for 1.12.2
2. Download the `.jar` file from [CurseForge](https://legacy.curseforge.com/minecraft/mc-mods/enchanting-infuser-backport)
3. Place it in your `.minecraft/mods` folder
4. Launch the game and enjoy!
